    • Girly Eatz

    Overview of the System

    Girly Eatz is an academic group project aimed at developing a user-friendly website with a comprehensive list of restaurants near iACADEMY. The website provides important information about each restaurant, such as its distance from iACADEMY, to help students, professors, and visitors identify nearby meal alternatives. The initiative caters to the iACADEMY community and visitors searching for local meal alternatives, providing a curated list of eateries with distance measurements. The technical stack is comprised of HTML and CSS for the frontend, Python for backend operations and data administration, and PhpMyAdmin for managing the restaurant information database. Girly Eatz will make it easier to identify nearby meal alternatives, serve as a great resource for daily convenience, and improve the overall campus experience.
